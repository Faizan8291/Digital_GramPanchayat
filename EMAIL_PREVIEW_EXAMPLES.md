## ğŸ’¡ CUSTOMIZATION OPTIONS

Want to customize the emails? Edit these methods in `EmailService.java`:

1. **Change Colors/Styling**: Modify the `<style>` section in:
   - `buildApprovalEmailHtml()`
   - `buildRejectionEmailHtml()`

2. **Change Text**: Modify the HTML content strings

3. **Add Logo**: Include an image tag:
   ```java
   "<img src='https://your-domain.com/logo.png' alt='Logo' style='width:150px;'/>"
   ```

4. **Add Footer Links**: Add anchor tags to footer section

5. **Change From Name**: Update in `application.properties`:
   ```properties
   app.mail.fromName=Your Custom Name
   ```
